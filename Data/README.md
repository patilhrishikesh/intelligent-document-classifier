# Dataset Description

This directory contains raw, unprocessed text documents used for training
the Intelligent Document Classifier.

## Document Categories
- resume   : Job resumes and CV-like documents
- invoice  : Billing and payment-related documents
- report   : Articles, reports, and formal written documents

## Notes
- All documents are stored as plain text (.txt)
- No preprocessing is applied at this stage
- Labels are inferred from folder names
